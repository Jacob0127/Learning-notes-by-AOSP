## Android

1. FP的

   Android系统的构建指纹。
   系统构建标识符: ro.system.build.fingerprint 是 Android 系统用来唯一标识特定构建版本的字符串
   软件版本区分: 用于区分不同的软件构建版本，特别是在认证和更新场景中

2. GMS：Google Mobile Service，谷歌移动服务

3. GTS：GMS Test Suite，GMS测试套件
4. RRO：Runtime Resource Overlay，运行资源覆盖
5. SRO：Static Resource Overlay，静态资源覆盖
6. SMR：Safety Maintenance Release，专用于安全维护的版本
