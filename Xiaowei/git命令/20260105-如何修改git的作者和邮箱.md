# 如何修改

## 首先使用命令查看当前作者和邮箱

### 命令

```bash
git config --global user.name
```

```bash
git config --global user.email
```



![image-20260105095955325](C:\Users\admin38\AppData\Roaming\Typora\typora-user-images\image-20260105095955325.png)

## 再使用命令进行修改

### 命令

```bash
git config --global user.name "修改的名字"
```

```bash
git config --global user.email "邮箱地址"
```
