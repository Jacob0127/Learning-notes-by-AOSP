# 问题

在登录、注册账号的时候，需要输入密码。而为了不让密码明文保存在数据库中，我们需要对密码进行加密，一开始我是直接使用**盐+md5**的方式来对密码进行加密。

但是这样的加密方式很容易被破解，所以我们需要更好的加密方式。

# 解决方法

使用**BCrypt**来对密码进行加密，这个能够很难被暴力破解

## 具体实现

我是使用了**hutool**工具包中自带的工具，使用方法如下：

```java
// ======== 注册时 ========
// 密码加密
String encode = BCrypt.hashpw(password);
// ======== 登录时 ========
// 密码匹配
boolean result = BCrypt.checkpw(password, user.getPassword());
if (!result) {
	throw new BusinessException(ErrorCode.PARAMS_ERROR, "密码错误");
}
```
