# 问题：登录时，存储token进入redis发现存入的value前面有很多空格

# 原因：Redis方法使用错误

# 错误方法

```java
redisTemplate.opsForValue().set(uuid,text,60);
```

这个方法的`60`并不是过期时间，而是偏移量

![img](https://i-blog.csdnimg.cn/blog_migrate/4b9b871494185ccc3213ad18345e30a4.png)

# 正确方法

```java
redisTemplate.opsForValue().set(uuid,text,60,TimeUnit.SECONDS);
```
