

## **一、基本查找：grep**

### **1. 在当前目录递归查找**

```
grep -r "要查找的内容" .
```

- `-r`：递归搜索
- `.`：当前目录

### **2. 在指定目录查找**

```
grep -r "内容" /path/to/directory/
```

### **3. 查找时显示行号**

```
grep -n "内容" 文件名
grep -rn "内容" 目录/
```

### **4. 忽略大小写**

```
grep -ri "content" .
```

- `-i`：忽略大小写

## **二、高级查找**

### **1. 只显示文件名（不显示内容）**

```
grep -rl "内容" .
```

## **三、按文件类型过滤**

### **1. 只搜索特定扩展名**

```
grep -r "内容" --include="*.java" .
grep -r "内容" --include="*.cpp" --include="*.h" .
```

### **2. 排除某些文件/目录**

```
grep -r "内容" --exclude-dir=".git" .
grep -r "内容" --exclude="*.o" --exclude="*.so" .
```

### **3. 使用 find + grep 组合**

```
find . -name "*.java" -exec grep -l "内容" {} \;
```
