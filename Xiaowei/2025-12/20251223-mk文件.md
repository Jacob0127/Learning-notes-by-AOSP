# 一.PRODUCT_AAPT_CONFIG

```
PRODUCT_AAPT_CONFIG := xhdpi hdpi mdpi normal
```

- 定义 Android 资源编译时支持的屏幕密度配置。
- xhdpi、hdpi、mdpi、normal 表示不同分辨率的资源目录，确保应用在多种设备上正确显示。

# 二.PRODUCT_AAPT_PREF_CONFIG

```
PRODUCT_AAPT_PREF_CONFIG := xhdpi
```

- 指定默认优先使用的屏幕密度为 xhdpi。
- 编译时会优先选择该密度下的资源文件（如图片、布局）。

# 三.显示与图形相关配置

## PRODUCT_PROPERTY_OVERRIDES

```
PRODUCT_PROPERTY_OVERRIDES += \
    ro.sf.lcd_density=240 \
    ro.vendor.sf.lcd_width=48 \
    ro.vendor.sf.lcd_height=85
```

### 1. ro.sf.lcd_density

```
ro.sf.lcd_density=240
```

- 设置 LCD 密度为 240 dpi，影响 UI 元素缩放。

### 2.ro.vendor.sf.lcd_width

```
ro.vendor.sf.lcd_width=48
```

- 设置屏幕宽度为 48mm（单位：毫米）。

### 3.ro.vendor.sf.lcd_height

```
ro.vendor.sf.lcd_height=85
```

- 设置屏幕高度为 85mm（单位：毫米）。
