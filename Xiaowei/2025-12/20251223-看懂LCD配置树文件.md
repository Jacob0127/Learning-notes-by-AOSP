# 一. 文件用途

配置 xxx LCD面板的初始化参数、时序、复位序列等

# 二. 每个代码的含义

## fragment、target-path、overlay

```
/ {
	fragment {
		target-path = "/";
		__overlay__ {
```

- 定义一个设备树片段（fragment），用于覆盖或扩展主设备树。
- target-path = "/" 表示目标路径为根节点。
- overlay 是 Linux 内核中用于动态加载设备树补丁的关键字。

## lcds

```
			lcds {
				lcd_gc9503_mipi: lcd_gc9503_mipi {
```

- 定义一个名为 lcd_gc9503_mipi 的 LCD 节点，作为屏幕控制器的配置入口。

## sprd

```
     sprd,dsi-lane-number = <2>;
```

- 表示sprd（这里代表的是展锐芯片平台）的私有属性，是一个前缀

## 基本工作模式与接口配置

### 1. dsi-work-mode

```
prd,dsi-work-mode = <1>;
```

- dsi-work-mode = <1>：使用 DSI 模式（通常为视频模式）。

### 2. dsi-lane-number

```
sprd,dsi-lane-number = <2>;
```

- dsi-lane-number = <2>：使用 2 条数据通道传输数据。

### 3. dsi-color-format

```
sprd,dsi-color-format = "rgb888";
```

- dsi-color-format = "rgb888"：颜色格式为 RGB888（每通道 8 位）。

## PHY时钟设置

### 1. phy-bit-clock

```
sprd,phy-bit-clock = <460000>;
```

- phy-bit-clock = <460000>：MIPI PHY 的比特时钟频率为 460 kHz（单位 Hz）。

### 2. phy-escape-clock

```
prd,phy-escape-clock = <20000>;
```

- phy-escape-clock = <20000>：逃逸时钟频率为 20 kHz，用于低速通信。

## 屏幕物理尺寸

### 1.width-mm

```
sprd,width-mm = <68>;
```

- 宽68mm

### 2.height-mm

```
sprd,height-mm = <121>;
```

- 高121mm

## ESD检查配置（电荷敏感检测）

### 1.esd-check-enable

```
sprd,esd-check-enable = <0>;
```

- esd-check-enable = <0>：禁用 ESD 检测。
- esd-check-enable = <1>：启用 ESD 检测。

### 2.esd-check-mode

```
sprd,esd-check-mode = <1>;
```

- esd-check-mode = <1>：若启用，使用寄存器读取方式检查。
- 这里<1>表示启用寄存器读取方式检查
- 最终还需要看esd-check-enable开启ESD检测，也就是esd-check-enable = <1>

### 3.esd-check-period

```
esd-check-period = <1000>;
```

- 检查周期为1秒

## 复位序列控制

### 1.reset-on-sequence

```
sprd,reset-on-sequence = <1 20>, <0 50>, <1 120>;
```

- 上电复位：
  先拉高复位引脚 20ms → 拉低 50ms → 再拉高 120ms。

### 2.reset-off-sequence

```
sprd,reset-off-sequence = <1 20>;
```

- 拉高复位引脚 20ms 后释放。

## 初始化命令

### 1.sprd,initial-command

```
sprd,initial-command = [
	...
];
```

- 包含一系列向 LCD 控制器写入的寄存器命令，用于完成屏幕初始化。
- 格式说明：
  - 每组命令以 39 或 23 开头，表示命令长度和类型。
  - 如 39 00 00 06 F0 55 AA 52 08 00 表示：
  - 6 字节命令，地址 F0，数据 55 AA 52 08 00。

- 这些命令来自 LCD 驱动芯片（GC9503）的数据手册，用于配置伽马曲线、亮度、对比度、显示模式等。

## 睡眠控制命令

### 1.sleep-in-command

```
sprd,sleep-in-command = [...];
```

- 进入睡眠：执行一系列命令关闭背光、停止刷新等操作。

### 2.sleep-out-command

- 退出睡眠：执行 13 78 00 01 11 和 13 64 00 01 29，分别对应 DISP_ON 和 DISPLAY_ON 命令，唤醒屏幕。

## 显示时序

### 1.display-timings

```
display-timings {
};
```

- 屏幕的时序

### 2.native-mode

```
native-mode = <&gc9503_timing0>;
```

- 表示该显示屏的“原生分辨率与时序”。

### 3.clock-frequency

```
clock-frequency = <32000000>;
```

- 像素时钟：32MHz

### 4.hactive和vactive

```
hactive = <480>;
vactive = <960>;
```

- 分辨率：480 × 960

### 5.hback-porch、hfront-porch、hsync-len

```
hback-porch = <24>;
hfront-porch = <24>;
hsync-len = <10>;
```

- 水平同步信号：
  - 前沿 24，后沿 24，同步脉冲宽度 10

### 6.vback-porch、vfront-porch、vsync-len

```
vback-porch = <16>;
vfront-porch = <16>;
vsync-len = <8>;
```

- 垂直同步信号：
  - 前沿 16，后沿 16，同步脉冲宽度 8
