# Linux中的权限及相关命令

## 查看权限

```
ls -l(或者ll)
```

### 输出示例：

```
-rwxr-xr-- 1 user group 4096 Oct 10 12:00 file.txt
```

### -rwxr-xr--的含义：

第一位：文件类型（-：普通文件，d：目录，l：符号链接，c：字符设备，b：块设备）

第二位到第四位（三位）：所有者权限（rwx：读、写、执行）

第五位到第七位（三位）：所属组权限（r-x：读、执行）

后三位：其他用户权限（r--：只读）

- r：读
- w：写
- x：执行



## 修改权限（chmod）



### 修改权限的两种方式：

#### 符号方式

- u ：文件所有者，g ：所属组，o ：其他用户，a ：所有用户。

- +：添加权限，- ：移除权限，=： 设置为指定权限。
- 示例：

```
chmod u+x file.txt # 给文件所有者添加执行权限
chmod g-w file.txt # 移除所属组的写权限
chmod o=r file.txt # 设置其他用户为只读权限
```

#### 数字方式：通过三位八进制数字表示权限

- 7：读、写、执行（4+2+1）
- 6：读、写（4+2）
- 5：读、执行（4+1）
- 4：只读（4）
- 示例：

```
chmod 755 file.txt # 文件所有者有所有权限，组和其他用户有读、执行权限
chmod 644 file.txt # 文件所有者有读写权限，组和其他用户只有读权限
```



## 修改文件所有者或组（chown和chgrp）

- chown：更改文件的所有者
- chgrp：更改文件的所属组
- 示例：

```
chown user file.txt # 将文件所有者改为 user
chgrp group file.txt # 将文件所属组改为 group
chown user:group file.txt # 同时修改所有者和所属组
```
