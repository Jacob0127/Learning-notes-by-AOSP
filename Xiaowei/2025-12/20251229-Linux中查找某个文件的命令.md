# 基本语法

```bash
find [搜索路径] [选项] [条件]
```

# 常用示例

## 按文件名查找

```bash
find /home -name "filename.txt"          # 精确匹配
find / -name "*.conf"                    # 查找所有.conf文件
find . -iname "readme"                   # 忽略大小写
```

## 按类型查找

```bash
find /var/log -type f -name "*.log"      # 只找文件(-f)
find /home -type d -name "projects"      # 只找目录(-d)
find /dev -type l                        # 只找符号链接(-l)
```

## 按时间查找

```bash
find / -mtime -1                         # 过去24小时内修改的
find /tmp -amin +30                      # 30分钟前访问的
find /var -newer reference.txt           # 比某个文件更新的
```

## 按大小查找

```bash
find / -size +100M                       # 大于100MB
find ~ -size -10k                        # 小于10KB
find . -size +50M -size -100M            # 50MB到100MB之间
```

## 组合条件与执行操作

```bash
find /home -name "*.tmp" -delete         # 找到后删除
find . -name "*.java" -exec grep -l "main" {} \;  # 找到后执行命令
find / -type f -perm 777 -print          # 找权限777的文件
```
