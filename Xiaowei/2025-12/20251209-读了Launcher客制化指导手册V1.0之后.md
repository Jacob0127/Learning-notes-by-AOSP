#### 1.3 预置图标

1. 如何查看是否存在Partner App：

   方式一（windows cmd命令行中）：

   ```
   adb logcat | findstr "UNISOC-LAUNCHER3"
   ```

   方式二（powershell中）：

```
adb logcat | Select-String "UNISOC-LAUNCHER3"
```

​		方式三（Linux中）：

```
  adb logcat | grep "UNISOC-LAUNCHER3"
```

 2. 根据包名查找对应的配置文件

    如果修改配置文件无效，有可能是被Launcher Partner机制替换
