# 找到已安装应用的包名

```bash
# 命令
adb shell pm list packages | findstr -i <大概的模块名>

# 导出所有用户安装的应用
adb pm list packages
```

# 根据包名找到对应apk路径

```bash
# 命令
adb shell pm path <包名>
```

# 根据apk路径提取apk到电脑

```bash
# 命令
adb pull <源文件路径> <目标路径>
```
