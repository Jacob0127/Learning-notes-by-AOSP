# 定义变量的语法

```bash
# 变量=值
your_name="zhangsan"
```

> 注意：等号两边不能有空格

# 变量的使用

在使用变量之前，需要先定义该变量，定义变量之后，使用的时候，只需要`$变量名`或者`${变量名}`即可

```bash
# 定义变量
your_name="zhangsan"
# 使用变量
echo $your_name
echo ${your_name}
```

> 上面两种使用方法都可以，但是推荐使用`${变量名}`

# 修改变量

已定义的变量，可以被重新定义

```bash
# 定义变量
your_name="zhangsan"
# 重新定义
your_name="lisi"
```

# 只读变量

使用`readonly`命令可以将变量定义为只读变量，只读变量无法修改。

```shell
readonly your_name
```

# 删除变量

使用`unset`命令可以删除变量，但是不能删除只读变量

```shell
unset your_name
```
