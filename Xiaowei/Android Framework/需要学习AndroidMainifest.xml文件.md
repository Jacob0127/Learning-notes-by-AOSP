# 问题：我需要完全看懂该文件吗

不需要完全看懂，但需要掌握核心内容。

---
  常见场景

  场景1：让一个 Activity 成为桌面

  <activity android:name=".Launcher">
      <intent-filter>
          <action android:name="android.intent.action.MAIN" />
          <category android:name="android.intent.category.HOME" />  <!-- 关键！ -->
      </intent-filter>
  </activity>
  你只需要知道：加上 category.HOME 才能成为桌面。

  场景2：声明权限

  <uses-permission android:name="android.permission.WRITE_SETTINGS" />
  你只需要知道：加权限就写这个标签。

  场景3：应用组件无法启动

  <activity android:name=".MyActivity"
      android:exported="false" />  <!-- false = 外部无法启动 -->
  你只需要知道：exported="false" 会导致外部无法启动这个 Activity。

              ├── <receiver>     ← 广播接收者
              └── <provider>      ← 数据提供者
