# Android.mk文件基本格式

## 常用语法

1. 每个Android.mk文件**必须**以定义`LOCAL_PATH`为开始

   ```makefile
   LOCAL_PATH:=$(call my-dir)
   ```

2. `CLEAR_VERS`变量由`Build System`提供，并指向一个指定的 **GNU Makefile**，由它负责清理很多 `LOCAL_xxx`

   ```makefile
   include $(CLEAR_VARS)
   ```

3. 模块名

   ```makefile
   LOCAL_MOUDLE:=Demo
   ```

4. apk名

   ```makefile
   LOCAL_SRC_FILES:=$(LOCAL_MOUDLE).apk
   ```

   > 一般引用的是模块名

5. user eng tests optional

   ```makefile
   LOCAL_MOUDLE_TAGS:=optional
   ```

   > user版本
   >
   > eng版本 
   >
   > tests版本 
   >
   > option所有版本都编译（默认）

6. 指定模块的类型，可不用定义

   ```makefile
   LOCAL_MOUDLE_CLASS:=APPS
   ```

   > APPS：apk
   >
   > JAVA_LIBRAYIES：java libraryies库
   >
   > SHARED_LIBRAYIES：类似于java的libraryies库
   >
   > EXECUTABLES：

7. 签名

   ```makefile
   LOCAL_CERIFICATE:=PRESIGNED
   ```

   > 你要编译的apk或者源码生成的apk，要使用什么样的签名
   >
   > platform：平台签名（安卓系统源码的签名），用该签名，该apk就变成了系统应用
   >
   > PRESIGNED：代表已经签名过了
   >
   > shared：数据共享的签名，可以共享联系人、电话之类的

8. 预置位置

   ```makefile
   LOCAL_PRIVILEGED_MOUDLE:=false
   ```

   > true：预置在system/priv-app中
   >
   > false：预置在system/app中
   >
   > priv-app的权限比app中的高，比如要使用系统限制的权限（比如：android.permission.WRITE_SECURE_SETTINGS)，我们就需要把程序安装到/system/priv-app中

   ```makefile
   LOCAL_SYSTEM_EXT_MOUDLE := flase
   LOCAL_MOUDLE_SUFFIX := $(COMMON_ANDROID_PACAGE_SUFFIX)
   include $(BUILD_PREBUILT)
   ```

   > true：预置在system_ext
   >
   > false：预置在system中

## 其他语法

1. 使其他模块不参与编译

   ```makefile
   LOCAL_OVERRIDES_PACKAGES := messaging Mms Message
   ```

   > 在apk编译时，会使 `messaging`、`Mms`、`Message`都不会编译

2. 指明所需要的库

   ```makefile
   LOCAL_OPTIONAL_USES_LIBRARIES := org.apache.http.legacy androidx.window.extensions
   ```

3. 定义资源文件的命名模式，用于支持不同屏幕密度下的资源适配。比如：

   ```makefile
   LOCAL_DPI_FILE_STEM := $(LOCAL_MOUDLE)_$(my_src_arch)_%.apk
   ```

   > `LOCAL_DPI_FILE_STEM`被定义为`ic_launcher`，而屏幕密度变量`LOCAL_DENSITY`的值为`hdpi`，那么根据这些变量生成的资源文件名将是`ic_launcher_hdpi.pn`

4. 为了适应不同屏幕密度的设备，可以通过提供不同密度的资源文件。这些资源文件以不同的命名约定存储在项目中，并通过`LOCAL_DPI_VARIANTS`指定哪些密度变种应该被编译和包含在最终的应用中。

   ```makefile
   LOCAL_DPI_VARIANTS := xxxhdpi xxhdpi xhdpi hdpi mdpi
   ```
