## 配置示例

```yaml
systemLog:
  destination: file
  path: "D:/mongodb/log/mongod.log" # 日志文件路径（用/或\避免转义）
  logAppend: true # 日志追加模式
storage:
  dbPath: "D:/mongodb/data" # 数据存储路径
journal:
  enabled: true # 启用日志持久化
net:
  port: 27017 # 服务端口
```

## 启动命令

```bash
mongod -f <配置文件路径>
# 或者
mongod --config <配置文件路径>
```
