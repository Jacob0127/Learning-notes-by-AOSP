# 是什么

> 类似于关系型数据库（MySQL）中的表
>
> 可以显式的创建，也可以隐式的创建

# 显式创建集合

## 语法

```
db.createCollection(name)
```

> name为集合的名称

# 隐式创建集合

当向集合中插入文档时，如果该集合不存在，会自动创建集合

# 查看集合
