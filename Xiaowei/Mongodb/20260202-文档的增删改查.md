# 什么是文档

文档数据结构跟JSON基本一样
所有存储在集合中的数据都是BSON格式

# 插入文档

## 单个文档的插入

### 语法

```bash
db.集合名称.insert(
	{
		"key值(name/age/number...)": "value值(张三/14岁/123...)",
		"key值(name/age/number...)": "value值(张三/14岁/123...)"
	}
)                                                                    
```

> 注意：
>
> 老版本的Mongodb也可以使用`db.集合名称.save()`的方法来插入文档
>
> 但是现在新版本已经将`save`废除掉了

# 文档的更新

## 语法

```bash
db.集合名称.updateOne(query, update)
# query:查询条件
# update:更新条件 update中 一定要使用 $set: 操作符
```

> `update`方法已经已弃用，需要用`updateOne`，`updateMany`，`bulkWrite`
>
> - `updateOne()` - 更新第一个匹配的文档
> - `updateMany()` - 更新所有匹配的文档
> - `replaceOne()` - 替换整个文档（不需要操作符）

### 示例命令

```bash
db.users.update({username: 'john_doe'},{$set: {age: 18})
```

# 删除文档数据

## 语法

```bash
db.集合名称.remove({条件}) 
# 如果不写条件，就是全删除，慎用！！！
# 写了条件，满足条件的都会删除
```
