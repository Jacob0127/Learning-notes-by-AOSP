# 语法

```bash
# $and
# 查询同时满足query1，query2的数据
db.集合名称.find({$and:[ {query1}, {query2} ]})

# $or
# 查询满足query1，query2其中一个的数据
db.集合名称.find({$or:[ {query1}, {query2} ]})
```
