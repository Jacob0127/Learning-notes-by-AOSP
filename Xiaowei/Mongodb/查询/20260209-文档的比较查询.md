# 语法

```bash
db.集合名称.find({字段:{$gt:value}}) # $gt:大于
db.集合名称.find({字段:{$lt:value}}) # $lt:小于
db.集合名称.find({字段:{$gte:value}}) # $gte:大于等于
db.集合名称.find({字段:{$lte:value}}) # $lt:小于等于
```
