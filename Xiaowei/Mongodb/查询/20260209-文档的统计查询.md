# 统计查询

## 语法

```bash
db.集合名称.count(query, options)
```

## 参数：

| Parameter | Type     | Description                  |
| --------- | -------- | ---------------------------- |
| `query`   | document | 查询选择条件                 |
| `options` | document | 可选。用于修改计数的额外选项 |
